# Literature Review Notes

## Core Themes

- SDN controller placement problem formulations (k-median, k-center, facility location).
- Effects of distributed controllers on delay and fault tolerance.
- Heuristic, metaheuristic, and learning-based placement strategies.

## Typical Baseline Families

- Random and degree-based heuristics.
- Greedy farthest-first variants.
- Clustering approaches (e.g., k-means over topology coordinates).

## AI/Optimization Families

- Genetic algorithms and evolutionary optimization.
- Reinforcement learning for adaptive placement decisions.
- Hybrid approaches combining heuristics and learning.

## Evaluation Dimensions

- Average and worst-case switch-to-controller latency.
- Controller load balance.
- Robustness under link/controller failures.
- Runtime overhead and convergence behavior.
