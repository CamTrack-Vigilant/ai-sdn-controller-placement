# SDN Textbooks

Store SDN textbooks and long-form references in this folder.

## Suggested Sub-Organization

- `fundamentals/` - SDN architecture, OpenFlow, control/data plane basics.
- `advanced/` - controller design, resilience, optimization, AI in networking.
- `notes/` - chapter summaries and definitions.

## How to Use Textbooks with Papers

1. Use textbooks to define terms and assumptions.
2. Use papers to capture current SotA and measured performance.
3. If a paper claim conflicts with textbook assumptions, flag it in your notes.
